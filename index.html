A personal fitness intelligence pipeline — unifying sleep, nutrition, and training data into one daily answer.

Public showcase page · full package source lives in a private repo during development

The Problem

Three things drive training results — and every app tracks only one of them in isolation.

Sleep

Recovery, hormone regulation, muscle repair

Nutrition

Fuel for training, protein for muscle synthesis

Training

The stimulus that drives adaptation

How It Works

Triad pulls all three sources together and tells you — based on your actual data — what your body needs today.

Step 1

Ingest

Zepp/Amazfit and Cal AI both write into Apple Health; Hevy connects via its official REST API. Three sources collapse into two integration points.

Step 2

Normalize

Sleep, nutrition, and workout records join by calendar date into a single daily_summary table in SQLite — the join hub for everything downstream.

Step 3

Engineer Features

Raw numbers become signals: sleep debt, recovery score, training load change, per-muscle-group fatigue.

Step 4

Recommend

Computed features plus a free-text goal ("build muscle and lose fat") go to an LLM, which reasons over both to produce a workout, calorie target, and recovery guidance for the day.

Installation & Quick Start

Triad ships as an installable Python package with a small public API. This is what running it actually looks like.

Install

pip install triad

Usage

import triad

# 1. Initialize the database (run once)
triad.setup()

# 2. Create your profile
triad.create_profile(
    name="Jonathan",
    age=24,
    weight_lbs=178,
    height_in=66,          # 5'6"
    sex="male",
    goal_text="build muscle and lose fat",
    target_sleep_hrs=8.0,
    activity_level="active"
)
# ✓ Profile saved
# ✓ TDEE target:    3002 cal/day
# ✓ Protein target: 178g/day

# 3. Sync your data
triad.sync(
    hevy_api_key="your-hevy-api-key",
    apple_health_path="path/to/export.xml",
    calai_pdf_path="path/to/calai_summary.pdf"
)

# 4. Get today's recommendation
triad.today()
